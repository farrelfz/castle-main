# CASTLE KPM UNJ

SPA berbasis Next.js App Router untuk showcase karya, event akademik, dan komunitas riset mahasiswa.

## Section Utama

- Hero
- About
- Features
- Events
- Timeline Kegiatan
- Publication
- Showcase
- Archives
- Testimonials
- Contact Us

## Jalankan Lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Build Production

```bash
npm run build
npm start
```
